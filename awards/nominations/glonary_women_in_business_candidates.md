# Workflow рекомендация

1. Открой каждую LinkedIn-ссылку → проверь, что профиль активен и есть фото
2. Если ссылка 404 / профиль закрыт → использую помеченные «(LinkedIn search needed)» — поиск делается через https://www.linkedin.com/search/results/people/?keywords=ИМЯ+КОМПАНИЯ
3. Из 3 кандидатов на каждую номинацию выберешь 1–2 для приглашения
4. Если нужно ещё кандидатов на конкретную номинацию — скажи какую, добью ещё 3–5 имён
