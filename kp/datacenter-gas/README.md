# Коммерческое предложение · Дата-центр на газогенерации

Финальное КП ASGard для продукта **«Дата-центр на собственной газогенерации»** во Владимирской области.

## Файлы

### Полная версия (7 страниц)
- `ASGard_KP_Datacenter_Gas.pdf` — **готовый PDF на 7 страниц** (A4 landscape) для подробной презентации.
- `kp.html` — исходник вёрстки полной версии.

### Краткая версия (1 страница)
- `ASGard_KP_Short.pdf` — **компактный PDF на 1 страницу** (A4 landscape) для быстрой рассылки в почте/мессенджерах.
- `kp-short.html` — исходник вёрстки краткой версии.

### Ассеты и референсы
- `assets/logo-asgard.svg` — фирменный логотип в SVG (масштабируется без потерь).
- `assets/logo-asgard.jpg` — растровая версия логотипа.
- `assets/weichai-side.jpg` — фото установки сбоку (для стр. 6 полной версии).
- `assets/weichai-3q.jpg` — фото 3/4 спереди (для стр. 4 полной и для краткой).
- `assets/weichai-front.jpg` — резервное фото.
- `assets/bg-pattern.jpg` — фирменный волнистый зелёный паттерн.

## Структура КП (7 страниц)

| № | Раздел | Назначение |
|---|---|---|
| 1 | Обложка | Заголовок, оффер, 3 KPI-плашки (3,5 ₽/кВт·ч · 320 кВт · 24 МВт), спец-условие |
| 2 | Площадка | Земля в собственности, магистральный газ ГОСТ, Владимирская обл. |
| 3 | Форматы участия | Вариант A (1 станция · наш контейнер) / Вариант B (3 станции · свой контейнер) |
| 4 | Состав и стоимость | Таблица CAPEX + CTA-баннер с тарифом |
| 5 | Что входит в тариф 3,5 ₽/кВт·ч | 6 пунктов сервиса + правая колонка с тарифом |
| 6 | Технические характеристики | Weichai · Baudouin 6m33 (4 блока таблиц) |
| 7 | Условия и контакты | Спецоффер на 5 МВт + таблица условий + контактная карточка |

## Дизайн-система

- **Палитра:** фирменный зелёный градиент ASGard (`#1F7A3D` → `#2BA84A` → `#4CB85F`), белый фон, тёмно-зелёный текст (`#0E2E1E`), серая периферия.
- **Шрифты:** системный гротеск (Inter / Manrope / Helvetica Neue) с фолбэком на Arial — гарантированно рендерится на любой машине.
- **Формат страницы:** A4 landscape (297 × 210 мм), без полей.

## Как пересобрать PDF из HTML

Полная версия:
```bash
cd kp/datacenter-gas
google-chrome --headless=new --disable-gpu --no-sandbox \
  --no-pdf-header-footer --no-margins \
  --print-to-pdf=ASGard_KP_Datacenter_Gas.pdf \
  "file://$PWD/kp.html"
```

Краткая версия:
```bash
cd kp/datacenter-gas
google-chrome --headless=new --disable-gpu --no-sandbox \
  --no-pdf-header-footer --no-margins \
  --print-to-pdf=ASGard_KP_Short.pdf \
  "file://$PWD/kp-short.html"
```

Для просмотра отдельных страниц как PNG:

```bash
pdftoppm -r 150 -png ASGard_KP_Datacenter_Gas.pdf preview
```

## Что можно поправить под клиента

В `kp.html` ищите следующие места и меняйте:

- **№ КП и дата** — на обложке (`КП № 0610-01`, `от 10.06.2026`).
- **Имя клиента / преамбула** — при желании можно добавить блок «Уважаемый …» в стиле существующего КП ASGard.
- **Контакты ответственного** — ФИО менеджера сделки в карточке контактов на стр. 7.
- **Цена / тариф** — если меняется коммерческая часть, правьте таблицу на стр. 4 и баннеры на обложке, стр. 4, стр. 5, стр. 7.

## Источники приёмов

- **Promminer GAS** — формат тех-характеристик, инфографика «почему наша локация», структура «состав сервиса».
- **DC Mining** — CTA-баннер тарифа, компактные таблицы условий, оформление «что входит».
- **FASTMINE** — 3 KPI на обложке, блок «форматы участия», dark-секции для «почему это важно».
- **Существующее КП ASGard** — палитра, логотип, тон голоса, юр-дисклеймер по ст. 435 ГК РФ.
